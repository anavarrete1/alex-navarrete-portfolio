# ⚽FIFA Soccer Database (SQL) 

## Overview 

- This project is a SQL Database designed to manage soccer leagues, teams, coaches, players, and federations. This project demonstrates database design, SQL query development, and relational modeling. 

## Features

- SQL
- MySQL Workbench
- League Management
- Team Management
- Player Records
- Foreign Key relationships

## Database Structure

- Federation
- League
- Team
- Player
- Coach

## Skills Demonstrated 

- Primary & Foreign Keys
- SQL Joins & Queries 
- Database Design
- Entity Relationship Modeling (ERD)

## Future Improvements 

- Views
- Expand player statistics 
- Team & player performance tracking
